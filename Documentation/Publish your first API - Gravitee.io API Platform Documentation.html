This guide walks you through the process of creating your first API by using the Web UI Portal. API Gateway Portal UI can be accessed using the following URL:

http://imsyafiq.xyz:8000

Chicago Employee Salary API : In this tutorial we will use the Chicago Employee Salary API to set up our first proxy API. Chicago Employee Salary API returns JSON-formatted data via the following URL : http://adt.imsyafiq.xyz:5000/departments

Create your API

  1. Login to http://imysyafiq.xyz:8000.

  2. Open the user menu (top right) and click Administration.

  3. Click (+). Create an API from scratch, Click(->)

  1. Give your API a name, a version, a description and a context path. Click Next

  1. Make sure that Backend is set to http://adt.imsyafiq.xyz:5000. Click Next.

  1. Give your Plan a name, set Security Type as API key, a description and enable Auto Validate Subscription. Click Next.

  1. AtDoc.Click Skip.

  1. At Deployment, Click Create and start the API

Your API is now ready to be use by your consumers. Let’s see how to do this : Use your first API

This quick start was just an overview to create your first API. To go further into API management detail you can take a look at the User Guide
